# Tank1990
Bil 211 dersi için dönem sonu ödevi (V2)
Tank1990 oyuncunun karargahını (eagle) korurken rakipleri ile kapıştığı ok tuşları (1. oyuncu) ve wasd (2. oyuncu) ile oynanabilen bir oyundur.
(V1 üzerine birkaç değişiklik yapılarak oyun düzeltilmeye çalışılmıştır. Hala eksiklikleri vardır)
(Yapıldığı dönem: 2024 Yaz)
